Amazon UX Design Intern Project

Summer 2014

The Project

During my UX Design Internship at Amazon, I was tasked with the ideation of a new browser extension that allows Prime customers to discover free media content. Working closely with my manager and other designers, I defined user scenarios and took the concept from initial sketches to final responsive prototypes.

I’m extremely excited about this project, but unfortunately I can’t disclose many details yet because the project is still in production. Come back soon for more details!
